# Network-Scanning-and-Mapping-Lab
This project focuses on identifying, analyzing, and documenting devices within a network using industry-standard scanning tools. The primary objective is to understand network structure, detect active hosts, discover open ports, and identify potential security vulnerabilities.

The lab was conducted using tools like Nmap and Wireshark on a controlled environment (such as Kali Linux). Various scanning techniques, including TCP SYN scans, UDP scans, and OS detection, were performed to map the network topology and identify running services.

During the project, IP ranges were scanned to detect live hosts, followed by port scanning to identify open and filtered ports. Service and version detection helped in understanding what applications were running on target systems. Packet analysis using Wireshark provided deeper insights into network traffic behavior.

The findings were documented, highlighting potential security risks such as exposed services, misconfigurations, and unnecessary open ports. Recommendations were provided to improve network security, including firewall configuration, service hardening, and regular vulnerability assessments.

This project enhanced practical knowledge of network reconnaissance, ethical hacking methodologies, and cybersecurity best practices.

Tools Used:
Kali Linux – Lab environment
Nmap – Network scanning & port analysis
Wireshark – Packet capture & traffic analysis
Netdiscover – Host discovery in LAN
Nessus – Vulnerability assessment
Zenmap – Network visualization

Outcome:
The project successfully mapped the network, identified open ports and services, and highlighted potential security risks. It improved practical skills in network reconnaissance, scanning techniques, and security analysis.
